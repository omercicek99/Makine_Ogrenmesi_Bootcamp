# Makine_Ogrenmesi_Bootcamp
TechIstanbul'un 12/2025 te düzenlediği bootcampde yaptığım  makine öğrenmesi algoritmaları, Linear Regression ve Random Forest kullanarak Sağlık Sigortası Maliyet
Tahmini yapan bir yazılım geliştirdim.
Sağlık Sigortası Fiyatlandırması (Supervised - Regression)
Konu: Sigorta Masrafı Tahmini Hikaye: "Bir sigorta şirketi, müşterilerine poliçe teklifi verirken
süreci otomatize etmek istiyor. Şu ana kadar uzmanlar manuel hesap yapıyordu ancak bu çok
yavaş. Şirket, kişinin yaşına, cinsiyetine, Vücut Kitle İndeksine (BMI) ve sigara içip içmediğine
bakarak yıllık sağlık masrafını tahmin eden bir yapay zeka modeli geliştirmenizi istiyor."
